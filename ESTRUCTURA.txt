FLUJO DE EJECUCIÓN
═══════════════════════════════════════════════════════════════════════════════

1. Program.cs → FrmHome.ShowDialog()
   │
2. FrmHome: Usuario selecciona figura del MenuStrip
   │
3. FrmHome → FrmAllShapes.LoadShape("NombreFigura")
   │
4. FrmAllShapes: 
   ├─ ShapeRegistry.GetShape() → obtiene instancia
   ├─ _currentShape.GetInputLabels() → inicializa TextBox dinámicos
   ├─ RedrawCanvas() → dibuja figura inicial
   │
5. Usuario ingresa parámetros y clic "Calcular":
   ├─ Obtener valores de TextBox
   ├─ _currentShape.SetParameters(valores)
   ├─ perimeter = _currentShape.CalculatePerimeter()
   ├─ area = _currentShape.CalculateArea()
   ├─ _currentShape.Draw(Graphics, bounds, brush, pen)
   ├─ Mostrar resultados
   │
6. Opciones adicionales:
   ├─ "Limpiar" → Reset
   ├─ "Color Relleno" → ColorDialog → Redibujar
   ├─ "Color Contorno" → ColorDialog → Redibujar
   └─ "Salir" → Close

═══════════════════════════════════════════════════════════════════════════════
PRINCIPIOS SOLID IMPLEMENTADOS
═══════════════════════════════════════════════════════════════════════════════

✓ S (Single Responsibility): Cada clase = 1 responsabilidad
✓ O (Open/Closed): Extensible sin modificar código existente
✓ L (Liskov Substitution): Cualquier IShape funciona en FrmAllShapes
✓ I (Interface Segregation): Interfaces específicas por grupo
✓ D (Dependency Inversion): Dependen de IShape, no de clases concretas
